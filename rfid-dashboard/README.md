# RFID Access Control Dashboard

Современный веб-дашборд для Arduino RFID системы контроля доступа.

## Запуск

```bash
npm install
npm run dev
```

Открой http://localhost:5173 в **Chrome** или **Edge** (Web Serial API не поддерживается в Firefox и Safari).

## Arduino

### Готовый скетч

Используй скетч `arduino/RFID_AccessControl.ino`:

- **Чтение** UID → `SCAN:A1B2C3D4`
- **Запись** блоков по команде из дашборда
- **Heartbeat** (PING каждые 8 сек) — стабильное соединение
- **Debounce** 2.5 сек — нет дублей при удержании карты
- **Повторная инициализация** MFRC522 при сбое

### Подключение

1. Залей `arduino/RFID_AccessControl.ino` через Arduino IDE
2. Пин LED: 2 (опционально)
3. MFRC522: RST=9, SS=10
4. В дашборде нажми «Подключить Arduino», выбери порт

**Если отключалось сразу:** раньше порт открывался, но чтение начиналось после паузы — буфер Serial переполнялся. Сейчас чтение идёт сразу после `open()`.

### Протокол «полная карта»

После строки `SCAN:UID` Arduino шлёт строки `META:КЛЮЧ=значение` и в конце `CARD_END`. В дашборде открывается блок **«Полная информация о карте»** (тип PICC, SAK, сырой UID, блоки сектора 0 для MIFARE Classic, страницы для Ultralight/NTAG).

### Запись на карту

1. Подключи Arduino к дашборду
2. В блоке «Зарегистрированные карты» введи метку (до 16 символов)
3. Нажми «Записать»
4. Приложи карту к ридеру в течение 15 секунд
5. Успех — 3 моргания LED, «✓ Записано» в дашборде

## Демо-режим

Без Arduino нажимай **«+ Демо»** в живой ленте для имитации сканирования.

Зарегистрированные тестовые карты: `A1B2C3D4`, `E5F6G7H8`, `I9J0K1L2`, `M3N4O5P6`.
